C is a general purpose programming language, which can be used to design any type of desktop based applications.
C language program is hardware independent.
For C language, the standard compilers can be used to compile and execute the program.
In the C programming language, we can use standard function like printf(), scanf() etc for output and input.
Readability modifications, bug fixing are very easy in a C language program. C language compilers generate operating system dependent executable files that can be run on the same operating system.
GCC (GNU Complier collection), Borland turbo C, Intel C++ compiler are some of the popular compilers which are used to compile, execute a C language program.
